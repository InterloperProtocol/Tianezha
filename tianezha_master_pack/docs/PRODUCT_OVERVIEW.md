# Product Overview

Tianezha is a no-wallet-connect, address-entry, simulation-first agentic software world where any wallet can become a profile, interact socially, join simulated markets, and participate in a hybrid futarchy system anchored by two real governance assets.

Tianezha nodes natively buy and sell compute, services, storage, and preservation across the mesh. Payment rails are adapters. Conway is optional.

Interface Assembly is like ARC for agents: the shell stays legible, the runtime stays portable, and the market state can move as a savegame bundle instead of being trapped in one node.
